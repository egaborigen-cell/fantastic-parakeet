# Project Blueprint

## Overview

This is a Flutter application that demonstrates how to interact with JavaScript code in a web environment. The app is integrated with the Yandex Games SDK, allowing users to view their player name, a leaderboard, and the environment language.

## Features

*   **Yandex Games SDK Integration:** The application initializes the Yandex Games SDK and can retrieve the player's name and the environment language.
*   **JavaScript Interop:** The application uses `dart:js_interop` to call JavaScript functions from Dart and vice-versa.
*   **Leaderboard:** The application can fetch and display a leaderboard from the Yandex Games SDK.
*   **Environment Language:** The application can retrieve and display the environment language from the Yandex Games SDK.
